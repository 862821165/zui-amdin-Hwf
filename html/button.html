按钮

提供日常项目中遇到的所有按钮,一般按钮都没有设置颜色可自行添加。

基本按钮

                             
                              
    .btn_common { border: 0; padding: 10px 20px; color: #ffffff; background-color: #FF6666; }

    .btn_arc { border: 0; padding: 10px 20px; color: #ffffff; background-color: #FF6666;border-radius: 5px; }

    .btn_round { border: 0; padding: 10px 20px; color: #ffffff; background-color: #FF6666; border-radius: 50px; }

自适应宽度按钮

                             
                              
    .btn_w_common { border: 0; padding: 10px 20px; color: #ffffff; background-color: #FF6666; display: block; width: 100%; margin: 5px 0; }

边线按钮

                             
                              
    .btn_solid_common { border: 1px solid #FF6666; padding: 10px 20px; color: #FF6666; background-color: transparent; }

    .btn_dashed_common { border: 1px dashed #FF6666; padding: 10px 20px; color: #FF6666; background-color: transparent; }
